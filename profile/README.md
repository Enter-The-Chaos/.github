## 📦 Installation Guide

### 🔽 CurseForge (Recommended)

1. Install the **CurseForge Launcher**
2. Search for **Enter The Chaos SMP**
3. Click **Install**
4. Press **Play** and enter the chaos ⚡

---

### 🛠️ Manual Install (Advanced)

1. Download the modpack `.zip` from CurseForge
2. Import into your launcher (CurseForge / Prism / MultiMC)
3. Allocate **6–8GB RAM recommended**
4. Launch and enjoy

---

## 🖥️ Server Setup

Run your own **Enter The Chaos SMP** server:

### ⚙️ Basic Setup

1. Download the **Server Files** from CurseForge
2. Extract into your server directory
3. Open `eula.txt` → set to `true`
4. Run the start script

---

### 🔧 Recommended Specs

* 💻 CPU: 4+ Cores
* 🧠 RAM: 8–12GB
* 🌐 SSD Storage (Highly Recommended)

---

### 🌍 Multiplayer Tips

* Use a **dedicated host** for best performance
* Keep mods & configs synced with clients
* Restart regularly for stability
